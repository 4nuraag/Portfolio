# Modern Portfolio

This is a modern, high-performance portfolio built with [Next.js](https://nextjs.org), Tailwind CSS, Framer Motion, and Three.js.

## Features

- **Performance**: Optimized with Next.js App Router and server components.
- **Visuals**: Aceternity UI background beams, custom shaders, and 3D elements.
- **Interactivity**: Smooth scrolling, infinite carousels, and responsive navigation.

## Deployment

The easiest way to deploy this Next.js app is to use the [Vercel Platform](https://vercel.com/new?utm_medium=default-template&filter=next.js&utm_source=create-next-app&utm_campaign=create-next-app-readme) from the creators of Next.js.

### Steps to Deploy from GitHub:

1.  Push your code to a GitHub repository (already done!).
2.  Go to [Vercel.com](https://vercel.com/signup) and sign up/login with GitHub.
3.  Click **Add New...** -> **Project**.
4.  Find your `Portfolio` repository and click **Import**.
5.  Click **Deploy**.

Vercel will automatically detect the Next.js framework and build your site. Every time you push to `main` on GitHub, Vercel will automatically redeploy your site.
